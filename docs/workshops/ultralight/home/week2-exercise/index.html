World Wide Time

Demos

  • Using specific times in a timezone: Is it time for Fruitful School?
  • Using an external API call: Morning Glory

Download the code from the Github Repository

Instructions

  1. Setup a website with deliberate “open” and “closed” hours, or with a specific response to a certain time
  2. Develop the site, using one of the demo templates as a basis

    You may also combine time with another form of input, either using other data available in the API, or combining results from another API call.

Steps

  1. First focus on the HTML & CSS— create the structure both on/off states
  2. Setup a data attribute on a containing element to trigger different CSS to be applied. A space ' ' betweeen two selectors sets up a parent — child relationship. Using this, we can set up conditional style rules in our css, making sure that certain things only apply when a parent has a particular attribute.
  3. Write your JavaScript code to setup your logic, that then triggers the updates to this data attribute.
  4. If you are using data in addition to working with the object, first try setting up the logic by simply storing the expected / desired JSON data in a variable. (See more about the json syntax on the JS reference page.) Then, use the fetch API to switch your data to a live source.
