Create Github Account

Create account on Github, if you don't have one already

Download Github Desktop Program

Download and install Github Desktop

Download the code from my repository

  1. Go to this repository on Github.

    This is a starter repository for your class homepage.
  2. Click the green "Code" button and under "Clone", choose "Download ZIP"

    Put this code somewhere safe for now, where you can access it soon.

Create your own repository

  1. Log into github.com and create a new repository using the link under the + icon in the upper right of the GitHub homepage.
  2. Name your repository "ultralight" or something similar.
  3. Once you've created it, click "Open in Desktop". Make sure to choose a careful or useful location for your local website files, perhaps in your "Sites" folder.

Put my code into your new site's folder

  1. Using Finder or your file navigator, find the download from my repository and unzip it.
  2. Drag these files and folders into your new site's folder, where you carefully saved it. For me, it's in my "Sites" folder and called "ultralight."
  3. If you put the files in the right place, when you open up "Github Desktop" program again, it should be noticing changes, or look visually different than before.

Open your local index.html in your web browser

When you open this index.html in your web browser, you should see that the url starts with "file:///". This means it's the local version (on your computer).

This is the index.html that is inside your "ultralight" folder on your computer.

Open the local index.html in your text editor

  1. Look at index.html in your text editor and try making somes small changes, like putting your name, favorite fruit, and favorite website in.
  2. Save your changes. Then, refresh the version of the site in your browser to see updates.

Open Github Desktop and see it notice changes

  1. Github Desktop should have some code highlighted in green (or red) to show what you deleted and added. If this looks good, write some "commit message" in the area where it says "Update index.html." You could say anything here, but it should be a description of what you changed or did.
  2. Press the button "Commit to main".
  3. In the upper bar, press the tab "Push origin".

Go to the page for this repository on github.com and notice updates

  1. Navigate to the page for your repository on github.com.

    should be something like: https://github.com/yourusername/ultralight.
  2. Notice changes that happened recently, if your push was made correctly.

Set up Github Pages for your site

Github Pages is essentially github's free hosting service for repositories. There is a max file size. Which works well, since this is the ultralight workshop!

  1. On your repository page on github.com, go to "Settings" in the upper bar.
  2. In the left menu, click "Pages"
  3. Under "Branch" select "Main" in the dropdown, and then press "Save"
  4. Wait a minute or two, and then your site should appear at:
    https://yourusername.github.io/ultralight/

Once your site is up, good job!

When your site is online, copy and paste the URL into our google doc.
